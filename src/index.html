Values

There are two types of values when it comes to JavaScript syntax. One is called fixed values, known as Literals. And the other is called variable values, known as Variables.

Literals

Numbers and Strings are the only literal values. Don't worry you can use numbers with or without decimals.

Strings are just regular text written between single or double quotes.

Variables

In JavaScript variables are used to store data types. To declare a variable in JavaScript you must use the following keywords let, const, var. Followed by an equal sign to assign a value.

Operators

JavaScript uses arithmetic operators to create values

  • + is equal to addition
  • - is equal to subtraction
  • * is equal to multiplication
  • / is equal to division
  • = is used as an assignment operator

Keywords

Keywords are retained words that are apart of the JavaScript language.

Some we already talked about like let, const and var.

Comments

Not all code in JavaScript gets executed. Comments are a great way to organize your code for yourself, or more readable for other programmers.

There are two ways to execute a comment

One is with back slashes "//"

Second is having your text between a back slash and asterisks "/* */"
